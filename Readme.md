# TheSudiptaVerse - Personal Portfolio

![Portfolio Screenshot](Will be updated soon)

This is the repository for my personal portfolio website, designed to showcase my skills, projects, and professional experience as a Full-Stack Developer with a focus on Machine Learning.

**[Live Demo](Will be updated soon)**

---

## ✨ Features

-   **Responsive Design**: Fully responsive layout that looks great on all devices, from mobile phones to desktops.
-   **Interactive Hero Section**: Includes a dynamic typing animation to introduce myself.
-   **Project Showcase**: A clean grid layout to display my key projects with descriptions, tech stacks, and links to their repositories.
-   **Skills Overview**: A categorized list of my technical skills across frontend, backend, ML, DevOps, and design.
-   **Downloadable CV**: A direct link for visitors to download my resume.
-   **Easy Navigation**: A fixed navigation bar that allows for smooth scrolling to different sections of the page.

---

## 🛠️ Tech Stack

This portfolio is built with a focus on simplicity and performance, using fundamental web technologies:

-   **HTML5**: For the structure and content of the site.
-   **CSS3**: For all styling, including the layout, animations, and responsive design.
-   **JavaScript**: For interactive elements like the typing animation.

---

## 🚀 How to Use

To get a local copy up and running, follow these simple steps.

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/itzsudipta/TheSudiptaVerse-Portfolio.git
    ```
2.  **Navigate to the project directory:**
    ```sh
    cd TheSudiptaVerse-Portfolio
    ```
3.  **Open `index.html` in your browser:**
    Simply open the `index.html` file in your favorite web browser to view the site.

---

## ✏️ Customization

You can easily customize this portfolio to make it your own:

-   **Content**: All text content, project details, and links can be modified directly in the `index.html` file.
-   **Styling**: Colors, fonts, and layout can be adjusted in the `style.css` file.
-   **Typing Animation**: The strings for the typing animation can be changed in the `script.js` file.
-   **Assets**: Replace the images in the `assets/` directory with your own, including your profile picture, project images, and CV.
